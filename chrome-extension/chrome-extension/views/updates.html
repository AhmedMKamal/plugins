Attack of the crabigator!
  • You can now spell out hard-for-the-recognizer words in WaniKani!
    1. If already in English mode (answering meanings) then just start spelling it out!
    2. If in Japanese mode, first say "えいご" then spell the reading as you would type it
      (eg. for しょうがつ say "s-h-o-u-g-a-t-s-u")
  • Added commands to switch between Japanese and English
  • Fixed bugs with general Japanese and WaniKani
